# Investment Allocator

A simple web application to visualize investment allocation based on your age and investment amount. The application uses **HTML, CSS, and JavaScript** with **Chart.js** to display a dynamic pie chart of your portfolio.  

---

## Features

- Enter your **investment amount** and **age**.
- Dynamically generates a **pie chart** showing allocation to:
  - Index Funds
  - Debt Funds
  - Real Estate / REITs
  - Gold
  - Cash (Emergency Fund)
- Shows **portfolio type** based on age:
  - Growth-Oriented Portfolio (<30 years)
  - Balanced Portfolio (30-40 years)
  - Conservative Portfolio (>40 years)
- Pie chart labels **appear inside each slice** with amount and percentage.
- **Legend / color indicators removed** for a clean look.
- Fully responsive for mobile and desktop.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- [Chart.js](https://www.chartjs.org/)
- [chartjs-plugin-datalabels](https://chartjs-plugin-datalabels.netlify.app/)
